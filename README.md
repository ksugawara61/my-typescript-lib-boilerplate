# my-typescript-lib-boilerplate

A boilerplate for TypeScript, Jest and library.

# Setup

```
$ yarn install
```

# Usage

## Run

```
$ yarn ts-node bin/index.ts
```

## Build

```
$ yarn build
```

## Lint

```
$ yarn lint
```

## Test

```
$ yarn test
$ npx http-server -o coverage/lcov-report # see also coverage report in your browser
```

## Format

```
$ yarn format
```
